# Servicios

## :material-api: APIs REST con Python

Diseño e implemento APIs con **FastAPI**: autenticación, validación con Pydantic, documentación OpenAPI automática, tests y despliegue. Ideal si tienes un producto web o móvil que necesita un backend a medida.

## :material-cog-transfer: Automatización de procesos

Convierto tareas manuales y repetitivas en flujos automáticos: **n8n**, scripts Python programados, integraciones entre APIs (Slack, Notion, Google Workspace, Airtable…). Menos copy-paste, más tiempo para lo importante.

## :material-spider-web: Web scraping y extracción de datos

Scrapers robustos y mantenibles con **BeautifulSoup**, **Playwright** o **Scrapy**. Manejo de anti-bots, paginación, deduplicación y entrega en el formato que necesites (CSV, base de datos, API).

## :material-database-arrow-right: Data engineering

Pipelines ETL/ELT con Python, Airflow o Prefect. Limpieza, transformación y carga de datos entre fuentes. Dashboards con **Streamlit** cuando el cliente necesita visualizar resultados sin complicaciones.

## :material-robot-outline: Integración de IA generativa

Experiencia aplicada en **RAG, LLM-as-a-judge, fine-tuning ligero y prompt engineering**. Integración de modelos (OpenAI, Azure OpenAI, modelos open-source) en productos reales con evaluación sistemática de calidad.
